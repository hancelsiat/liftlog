# LiftLog: Fitness Tracking Mobile Application

## Project Overview
LiftLog is a mobile application designed to help users track their fitness progress, manage workouts, and monitor exercise videos.

## Technologies Used
- Frontend: Flutter (Dart)
- Backend: Node.js, Express
- Database: MongoDB Atlas
- Deployment: Render (Planned)

## Prerequisites
- Flutter SDK
- Node.js
- MongoDB Atlas account

## Setup Instructions

### Backend Setup
1. Clone the repository
2. Navigate to backend directory
3. Install dependencies:
   ```
   npm install
   ```
4. Create .env file with:
   - MONGODB_URI
   - JWT_SECRET
   - Other configuration variables

### Mobile App Setup
1. Navigate to mobile directory
2. Get Flutter dependencies:
   ```
   flutter pub get
   ```

## Running the Application
- Backend: `npm start`
- Mobile: `flutter run`

## Deployment
- Backend: Render
- Database: MongoDB Atlas
